# Retrieval-Augmented Defense Project Page

Static project webpage for:

**Retrieval-Augmented Defense: Adaptive and Controllable Jailbreak Prevention for Large Language Models**

## Local Preview

Open `index.html` directly in a browser.

## GitHub Pages

1. Create a GitHub repository.
2. Push this folder to the repository.
3. In the repository settings, go to **Pages**.
4. Set **Source** to **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.

The page will be available at:

```text
https://YOUR_USERNAME.github.io/YOUR_REPOSITORY/
```

## Edit Before Publishing

- Replace the `Accepted Paper` label in `index.html` with the final venue name if desired.
- Replace the anonymous code/data link with the public repository URL when it is ready.
- Add extra assets such as a video, poster, slides, or paper PDF under `assets/`.
